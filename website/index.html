How it works

Three golden paths

The workflows Peep was designed around.

01

Open app → see it running

Open any Flutter or React Native project. Peep detects the platform, runs pub get or npm install, starts the dev server, and shows your app in the phone frame — in under 60 seconds.

02

Prompt → diff → preview

Type "Add a settings screen with dark mode toggle". The agent reads your project, writes the code, shows a unified diff, and waits for your approval. Accept → preview auto-refreshes.

03

Error → fix → green analyze

Build fails? The error appears in the panel with a file link. Click "Fix this" — the agent patches the file, runs flutter analyze, and confirms it's clean.
